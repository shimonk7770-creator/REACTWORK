# PRD - פרויקט חת"ת יומי

## מבוא

מטרה: לבנות אפליקציה ב-React המאפשרת למשתמשים לעקוב אחרי לימוד יומי של חומש, תהילים ורמב"ם, לקבל תזכורות, לצבור ניקוד עבור רצף יומי, ולפתור חידון פרשת השבוע.

## קהלי יעד

- לומדי חומש יומי, תהילים ורמב"ם
- משתמשים המעוניינים בתזכורות יומיות ובמעקב רצף למוטיבציה
- קהל דתי המחפש חוויית לימוד פשוטה וזמינה

## פונקציות מרכזיות

1. לוח לימוד יומי
   - תצוגת פרקי החומש, תהילים ורמב"ם לפי יום בחודש
   - אינטגרציה פשוטה להצגת פרקי תוכן מבוססי תוכנית יומי

2. תזכורות
   - התראות יומיות לביצוע לימוד
   - אפשרות להגדיר זמן קבוע

3. מערכת ניקוד רצף
   - ניקוד עבור ביצוע לימוד רציף
   - טיפול בסטיקות של רצף
   - הצגת סטטיסטיקות רצף חודשיות

4. חידון פרשת השבוע
   - שאלות טריוויה על פרשת השבוע
   - דירוג רמות קושי וניקוד
   - תצוגת תוצאות ותגובות

## דרישות פונקציונליות

- ניווט בין מסכי הבית, לוח לימוד יומי, חידון פרשה והיסטוריית ניקוד
- שמירת מצב משתמש מקומית ב-localStorage
- מסך הגדרות תזכורות
- עמוד פרטי על התוכנית ואיך לחזור על לימוד

## דרישות טכנולוגיות

- React (אפשר Vite/CRA)
- React Router למסלולים
- שמירה מקומית ב-localStorage
- CSS מודרני או ספריית UI קלה

## סדרי עדיפויות

- MVP ראשון: לוח יומי + ניקוד רצף + חידון פרשה בסיסי
- פיצ'רים מתקדמים: תזכורות מתוזמנות, סטטיסטיקות מורחבות, רמות קושי לחידון

## שאלות פתוחות

1. האם יש מקור נתונים קבוע לחומש, תהילים ורמב"ם (קבצים סטטיים, API, קישורים)?
2. האם רוצים רק שפה עברית, או גם תמיכה באנגלית?
3. האם יש צורך בהרשמה או שמירת משתמשים מרובים?
4. האם תזכורות צריכות לעבוד גם באפליקציה נייטיב או רק בדפדפן?
5. האם יש צורך בפיצ'ר "היום בחודש" במבנה חודשי קבוע או משתנה?
