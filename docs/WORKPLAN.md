# תוכנית עבודה - פרויקט חת"ת יומי

## שלב 1 - איפיון ו-PRD

- הכנת PRD בסיסי
- הגדרת מסלולים ודרישות ראשוניות
- שאלת שאלות למנכ"ל הפרויקט כדי להבהיר את ההיקף

## שלב 2 - בניית שלד הפרויקט

- יצירת פרויקט React חדש (Vite/CRA)
- הוספת React Router
- סידור מבנה תיקיות ראשוני
- דף בית + דף לוח לימוד יומי

## שלב 3 - מימוש לוח לימוד יומי

- תצוגת החומש, תהילים ורמב"ם לפי יום בחודש
- אחסון נתוני תוכנית יומי בקבצי JSON מקומיים
- ניווט בין תאריכים

## שלב 4 - מערכת ניקוד רצף

- חישוב ניקוד עבור ימים רצופים
- שמירת מצב הרצף ב-localStorage
- הצגת סטטיסטיקות רצף חודשיות

## שלב 5 - חידון פרשת השבוע

- יצירת מסך שאלות טריוויה
- הוספת רמות קושי וניקוד
- הצגת תוצאות ודירוג

## שלב 6 - תזכורות

- מסך הגדרות תזכורות יומיות
- שמירה מקומית של הגדרות
- הצגת התראות בתוך האפליקציה

## שלב 7 - שיפורים וחזות

- עיצוב חוויית משתמש ו-UI
- תמיכה במצב כהה אם רוצים
- בדיקות בסיסיות

## ניהול גרסאות

- כל שלב נשמר ב-commit נפרד
- לאחר שלב 2 נעשה push ל-remote
- בכל יום עבודה נוסיף commit עם תיאור ברור
