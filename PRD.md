# PRD - חת"ת יומי

## תמצית

פרויקט "חת"ת יומי" הוא אפליקציית React בעברית לניהול לימוד יומי של חומש, תהילים ותניא, עם חידוני פרשת השבוע, ניקוד רצף ותזכורות.

---

## מטרות עסקיות

- ✅ לאפשר למשתמש ללמוד תוכן יומי באופן פשוט ומעוצב
- ✅ לשפר שמירת רצף לימוד באמצעות ניקוד ומוטיבציה
- ✅ לתת חוויית חידון מעשירה על פרשת השבוע
- ✅ לשמור את כל השינויים והתקדמות המשתמש באופן מקומי ובגיט

---

## קהל יעד

- לומדי חומש, תהילים ותניא יומי
- קהל דתי המעוניין בכלי תרגול יומי וניקוד רצף
- משתמשים שאוהבים חוויית לימוד כיפית ועיצוב מודרני

---

## בחירות טכנולוגיות (Tech Stack)

| טכנולוגיה | גרסה | סיבת הבחירה |
|---|---|---|
| **React** | 18 | ספריית UI מובילה, hooks, קומפוננטים לשימוש חוזר |
| **Vite** | 5 | Build tool מהיר עם Hot Module Replacement — פיתוח נוח |
| **React Router** | v6 | ניווט client-side ללא טעינת דף, SPA אמיתי |
| **JavaScript ES2022** | — | פשטות — ללא TypeScript בשלב ה-MVP |
| **CSS3 ידני** | — | שליטה מלאה בעיצוב RTL ללא תלות בספריית UI חיצונית |
| **localStorage** | Web API | שמירת מצב מקומי ללא צורך ב-backend או DB |

### ספריות חיצוניות שנבחרו **לא** להשתמש בהן (ומדוע)
- **Redux / Zustand** — המצב קטן ומקומי, `useState` + `localStorage` מספיקים
- **Tailwind / MUI / Chakra** — עיצוב RTL עברי דורש שליטה מלאה ב-CSS
- **Axios / React Query** — אין API חיצוני בשלב הראשון
- **TypeScript** — MVP מהיר; ניתן להוסיף בהמשך

## ארכיטקטורה

### מבנה תיקיות

```
REACTWORK/
├── index.html               # נקודת כניסה HTML + lang="he" dir="rtl"
├── vite.config.js           # הגדרות Vite
├── package.json
└── src/
    ├── main.jsx             # ReactDOM.render + BrowserRouter
    ├── App.jsx              # Shell: header, nav, routes, reminder banner
    ├── index.css            # Global styles (RTL, dark theme, design system)
    ├── pages/
    │   ├── Home.jsx         # דף בית — hero card + כרטיסי תכונות
    │   ├── DailyBoard.jsx   # לוח יומי — חומש/תהילים/תניא + streak + score
    │   ├── Quiz.jsx         # חידון — שאלות רנדומליות + ניקוד לפי רמה
    │   └── Settings.jsx     # הגדרות — תזכורת יומית + סטטיסטיקה
    └── data/
        └── dailyContent.js  # תוכן: chumashByDayOfWeek + dailyContent[30]
```

### היררכיית קומפוננטים

```
App (BrowserRouter)
├── ReminderBanner           # באנר תזכורת (מותנה)
├── Header + Nav             # ניווט ראשי
├── Routes
│   ├── / → Home
│   ├── /daily → DailyBoard
│   ├── /quiz → Quiz
│   └── /settings → Settings
└── Footer
```

### ניהול מצב (State Management)

```
localStorage
├── reactwork-daily-state    { selectedDay, completedDays[], streak, score, lastCompletedDate }
├── reactwork-settings       { remindersOn, reminderTime }
├── reactwork-quiz-best      number (שיא ניקוד)
└── reactwork-reminder-seen  string (תאריך)
```

**גישה:** כל עמוד מנהל את המצב שלו עם `useState` + `useEffect` לקריאה/כתיבה ב-`localStorage`.  
אין state management גלובלי — המצב מחולק לוגית לפי דומיין (לוח יומי / חידון / הגדרות).

### זרימת נתונים (Data Flow)

```
dailyContent.js (static data)
        ↓
DailyBoard.jsx ←→ localStorage (streak, score, completedDays)
Quiz.jsx       ←→ localStorage (bestScore)
Settings.jsx   ←→ localStorage (reminderTime, remindersOn)
App.jsx        ←→ localStorage (reminder seen today)
```

## שימוש מרכזי

1. ✅ חוויית "לוח היום":
   - ✅ כל יום בחודש מציג חומש, תהילים ותניא
   - ✅ תוכן אמיתי — תהילים לפי יום בחודש, חומש לפי יום בשבוע, תניא במחזור חודשי
   - ✅ ניתן לשנות יום בחודש ולצפות בתוכן המתאים

2. ✅ מערכת רצף וניקוד:
   - ✅ המשתמש מסמן יום כמושלם
   - ✅ הקוד מוסיף ניקוד וניהול רצף
   - ✅ שמירת מצב ב-localStorage
   - ✅ הצגת ניקוד מצטבר וסטטיסטיקת רצף

3. ✅ תזכורות:
   - ✅ מסך הגדרות תזכורות יומיות
   - ✅ אפשרות לבחור זמן קבוע
   - ✅ התראות בתוך האפליקציה בעת כניסה

4. ✅ חידון פרשת השבוע:
   - ✅ שאלות טריוויה על תורה
   - ✅ ניקוד לפי רמת קושי
   - ✅ הצגת תוצאה ומשוב
   - ✅ בנק שאלות רנדומלי + שיא אישי

5. ✅ בונוס רצף (מיילסטונים):
   - ✅ 7 ימים ברצף = +50 נקודות בונוס 🎉
   - ✅ 14 ימים ברצף = +100 נקודות בונוס ⭐
   - ✅ 21 ימים ברצף = +150 נקודות בונוס 🌟
   - ✅ 30 ימים ברצף = +200 נקודות בונוס 🏆
   - ✅ באנר חגיגי עם אנימציה בהגעה לכל יעד
   - ✅ ספירה לאחור ליעד הבונוס הבא

7. ✅ פס התקדמות חודשי (Progress Bar):
   - ✅ פס גרפי עם גרדיאנט מוצג בלוח היומי ובהגדרות
   - ✅ סמני מיילסטון ב-7, 14, 21, 30 ימים על הפס
   - ✅ הצגת אחוז השלמה + ספירת ימים
   - ✅ אנימציית מילוי חלקה

8. ✅ אנימציות ומיקרו-אינטראקציות:
   - ✅ fadeInUp — כניסת עמוד וכרטיסים
   - ✅ completedPulse — ניצוץ ירוק בסימון יום
   - ✅ scoreBump — קפיצת ניקוד
   - ✅ streakBounce — קפיצת רצף
   - ✅ לחיצת כפתור: אפקט scale
   - ✅ feedbackSlide — כניסת פידבק חידון מימין

6. ✅ פרשת השבוע אוטומטית:
   - ✅ לוח פרשות מלא לשנת ה'תשפ"ו (45 שבתות)
   - ✅ חישוב אוטומטי לפי תאריך השבת הקרוב
   - ✅ הצגה בלוח היומי: שם פרשה + ספר + יום בשבוע
