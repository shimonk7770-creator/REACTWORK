# חת״ת יומי

אפליקציית React לניהול לימוד יומי של **חומש, תהילים ותניא** — עם ניקוד רצף, חידון פרשת השבוע ותזכורות יומיות.

🔗 GitHub: https://github.com/shimonk7770-creator/REACTWORK

---

## תכונות עיקריות

| תכונה | תיאור |
|---|---|
| 📖 **לוח יומי** | חומש לפי יום בשבוע, תהילים לפי יום בחודש, תניא במחזור חודשי |
| 🔥 **ניקוד ורצף** | סימון יום כהושלם + 10 נקודות, שמירת רצף ב-localStorage |
| 🧠 **חידון פרשה** | 6 שאלות רנדומליות מ-20, שלוש רמות קושי, שיא אישי |
| 🔔 **תזכורות** | הגדרת שעת לימוד + באנר תזכורת בכניסה לאפליקציה |
| 📅 **פרשת השבוע** | חישוב אוטומטי לפי תאריך — לוח פרשות ה׳תשפ״ו מלא |
| 🏆 **בונוס רצף** | מיילסטונים 7/14/21/30 ימים עם ניקוד בונוס ואנימציה |

---

## Tech Stack

- **React 18** + **Vite 5** — UI ו-build tool
- **React Router v6** — ניווט client-side (SPA)
- **CSS3 ידני** — עיצוב RTL מלא, dark theme, ללא ספריית UI
- **localStorage** — שמירת מצב מקומי ללא backend

---

## הפעלה מקומית

```bash
npm install
npm run dev
```

פתח בדפדפן: `http://localhost:5173`

---

## מבנה הפרויקט

```
src/
├── App.jsx              # Shell, ניווט, באנר תזכורת
├── index.css            # Global styles (RTL, dark theme)
├── pages/
│   ├── Home.jsx         # דף בית
│   ├── DailyBoard.jsx   # לוח יומי + streak + score
│   ├── Quiz.jsx         # חידון + randomization
│   └── Settings.jsx     # תזכורות + סטטיסטיקה
└── data/
    ├── dailyContent.js  # תוכן: תהילים (30 יום), תניא (30 יום), חומש (7 ימים)
    └── parashaData.js   # לוח פרשות ה׳תשפ״ו + getCurrentParasha()
```

## מפת האתר

```
/           → דף בית
/daily      → לוח לימוד יומי
/quiz       → חידון פרשת השבוע
/settings   → הגדרות ותזכורות
```
