# חת״ת יומי

אפליקציית React לניהול לימוד יומי של **חומש, תהילים ותניא** — עם ניקוד רצף, חידון פרשת השבוע ותזכורות יומיות.

🔗 GitHub: https://github.com/shimonk7770-creator/REACTWORK

---

## תכונות עיקריות

| תכונה | תיאור |
|---|---|
| 📜📗📕 **3 עמודי לימוד ייעודיים** | חומש, תהילים ותניא — כל אחד בעמוד נפרד עם עיצוב וצבע משלו |
| 📅 **סנכרון ליום העברי** | תהילים ותניא נבחרים לפי `HDate` (יום עברי בחודש), לא לפי תאריך לועזי — כולל טיפול ביום כ״ט/ל׳ מאוחד בחודש חסר |
| 🗓️ **לוח שנה עברי לחיץ** | לחיצה על יום בלוח מציגה מיד את התוכן של אותו יום בכל אחד משלושת העמודים |
| 🎬 **סרטון פתיחה** | סרטון YouTube מוטמע בדף הבית, מופעל אוטומטית ומושתק |
| 🔥 **ניקוד ורצף** | סימון יום כנלמד + 10 נקודות, שמירת רצף ב-localStorage (משותף לשלושת העמודים) |
| 🧠 **חידון פרשה** | בחירת רמת קושי (קל/בינוני/קשה) — כל רמה סבב נפרד של 10 שאלות מבנק אמיתי לפרשה (כולל שבתות כפולות), מעקב שאלות-שנשאלו לכל רמה, תגי הישג, אמירות עידוד וקונפטי, שיא אישי לכל רמה |
| 🔔 **תזכורות** | הגדרת שעת לימוד + באנר תזכורת בכניסה לאפליקציה |
| 📅 **פרשת השבוע** | חישוב אוטומטי לפי תאריך — לוח פרשות ה׳תשפ״ו מלא |
| 🏆 **בונוס רצף** | מיילסטונים 7/14/21/30 ימים עם ניקוד בונוס ואנימציה |
| 🎨 **עיצוב בהיר וצבעוני** | glassmorphism בהיר, פלטת זהב/סגול/כחול/טורקיז/ירוק, אנימציות מרובות |

## Tech Stack

- **React 18** + **Vite 5** — UI ו-build tool
- **React Router v6** — ניווט client-side (SPA)
- **@hebcal/core** — תאריך עברי (HDate), שמות חודשים, אירועים
- **CSS3 ידני** — עיצוב RTL מלא, ערכת נושא בהירה וצבעונית, ללא ספריית UI
- **localStorage** — שמירת מצב מקומי ללא backend

## מבנה הפרויקט

```
src/
├── App.jsx                    # Shell, ניווט, באנר תזכורת
├── index.css                  # Global styles (RTL, ערכת נושא בהירה)
├── pages/
│   ├── Home.jsx                # דף בית — כרטיסי נושא + סרטון פתיחה
│   ├── Chumash.jsx              # עמוד חומש (לפי יום בשבוע)
│   ├── Tehillim.jsx             # עמוד תהילים (לפי יום עברי בחודש)
│   ├── Tanya.jsx                 # עמוד תניא (לפי יום עברי בחודש)
│   ├── Quiz.jsx                  # חידון + 10 שאלות + מעקב שאלות-שנשאלו
│   └── Settings.jsx              # תזכורות + סטטיסטיקה
├── components/
│   ├── HebrewCalendarGrid.jsx    # לוח שנה עברי לחיץ (משותף לשלושת עמודי הלימוד)
│   ├── ProgressWidget.jsx        # כרטיס רצף/ניקוד משותף
│   └── TextViewer.jsx            # טעינת טקסט מ-Sefaria API
├── hooks/
│   └── useDailyProgress.js       # הוק משותף: יום נבחר + רצף/ניקוד (מבוסס HDate)
├── utils/
│   └── hebrewDate.js             # getHebrewDayInfo() — יום עברי בחודש, כולל חודש חסר
└── data/
    ├── dailyContent.js           # תוכן: תהילים (30 יום עברי), תניא (30 יום עברי), חומש (7 ימים)
    ├── parashaData.js            # לוח פרשות ה׳תשפ״ו + getCurrentParasha()
    └── parashaQuestions.js       # בנק שאלות לכל פרשה + טיפול בשבתות כפולות
```
