# REACTWORK

פרויקט חת"ת יומי – אפליקציה ב-React לניהול לימוד יומי של חומש, תהילים ותניא, עם ניקוד רצף וחידון פרשת השבוע.

## איך להפעיל

```bash
npm install
npm run dev
```

## רכיבים ראשוניים

- לוח יומי שמציג חומש, תהילים ותניא ליום בחודש
- מערכת ניקוד רצף שמורידה נתונים ל-localStorage
- חידון פרשת השבוע עם שאלות, רמות קושי וניקוד
